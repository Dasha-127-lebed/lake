🧠 Как работает нейросеть?

В отличие от линейной регрессии и дерева решений, нейросеть работает так:

1️⃣ Входные данные → взвешенная сумма
Каждый нейрон вычисляет: z = w₁·x₁ + w₂·x₂ + ... + wₙ·xₙ + b
где w — веса (важность признака), b — смещение.

2️⃣ Функция активации (ReLU)
Результат проходит через функцию: output = max(0, z)
Это добавляет нелинейность — сеть может учиться сложным зависимостям.

3️⃣ Многослойная архитектура
Данные проходят через слои: Вход → 50 нейронов → 30 нейронов → Выход
Каждый слой извлекает всё более абстрактные признаки.

4️⃣ Обучение (Backpropagation)
Сеть сравнивает предсказание с правильным ответом → вычисляет ошибку → корректирует веса в обратном направлении, чтобы уменьшить ошибку в следующий раз.

📋 Сравнение моделей:

Модель Логика работы Плюсы
Линейная регрессия Одна формула: y = w·x + b Проста, интерпретируема
Дерево решений Цепочка правил: if age > 30 then... Наглядно, не требует масштабирования
Нейросеть Многослойная сеть с нелинейными активациями Улавливает сложные, неочевидные зависимости

🔑 Главное: нейросеть сама учится находить важные признаки и их комбинации, тогда как в линейной регрессии мы задаём формулу вручную, а в дереве — правила ветвления.
